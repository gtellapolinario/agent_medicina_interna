# Skill bundle gerado

Este diretório contém uma skill pronta para uso:

- `SKILL.md`: instruções principais da skill
- `references/`: arquivos auxiliares de apoio
- `assets/`: template opcional de saída

## Uso
1. Revise o conteúdo do `SKILL.md`.
2. Ajuste `name` e `description` se quiser especializar ainda mais a ativação.
3. Mantenha o prompt principal mais enxuto e delegue o workflow desta tarefa para a skill.\n
